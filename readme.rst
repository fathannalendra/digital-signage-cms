📺 Digital Signage CMS & Live Report Analytics
Sistem Content Management System (CMS) berbasis web untuk mengendalikan jaringan layar Smart TV di berbagai showroom secara terpusat. Mengeliminasi penggunaan flashdisk manual dan memberikan laporan penayangan iklan yang tervalidasi secara langsung.

Fitur Utama:

☁️ Cloud Content Management: Atur antrean video (playlist) TV di berbagai cabang dari satu dashboard.

🛡️ Smart Device Filter: Mencegah data pelacakan kotor dengan mendeteksi dan memblokir traffic dari HP/Laptop admin.

📊 Background Tracker: JavaScript mencatat durasi putaran (loop) video dan mengirim log ke database secara real-time.

Tech Stack: CodeIgniter 3 | Vanilla JavaScript | MySQL
